# 80 Cereals Nutrition & Rating Analysis

This repository contains an exploratory data analysis of the **80 Cereals dataset** from Kaggle, focusing on the relationship between nutritional content and consumer ratings.

The goal is to understand which nutritional factors (like sugar, fiber, and protein) are associated with higher ratings and identify the “healthiest” cereals using a simple health score.

---

## 📂 Contents

- `cereal.xlsx` — Excel version of the dataset (originally from Kaggle)  
- `cereal_analysis.R` — R script with complete analysis:
  - Data loading  
  - Exploratory plots  
  - Correlation analysis  
  - Health score creation  
  - Linear regression modeling

---

## 📊 What You’ll Find

This project demonstrates:

- Data cleaning and inspection
- Visualization using **ggplot2**
- Correlation analysis with **corrplot**
- Feature creation (health score)
- Linear regression to investigate the influence of nutrients on cereal ratings

---

## 🛠️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/leeya068/80-cereals-analysis.git
   
2. Make sure you have R and the required packages installed:
   install.packages(c("tidyverse", "readxl", "corrplot"))

4. Place cereal.xlsx and cereal_analysis.R in the same folder.
5. Open the R script and run it to generate plots and results.
